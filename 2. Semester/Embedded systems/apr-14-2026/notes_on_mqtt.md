# MQTT
Today we will learn about MQTT, what it is, how to use it and much more!

## What is it?
Put simply MQTT is a protocol for transfering data
MQTT is a method that allows communication between many devices without needing them all to be on the same network 
and it can work with a database

It uses a Publish/Subscribe model for how it handles data and everything is handled by a broker 


it offers TLS encryption, uses usernames and passwords, can allow certs, and allows for easy backup

## Why use it?
MQTT is one of the most reliable and lightweight way to transfer data on systems with limited resourses, as well as allowing communication
between devices on different networks

## sending data 
When you send data you need to have a topic so you know what kind of information you are sending, and it will also have a QOA that 
indicates the quality of the data you are sending or reciving
