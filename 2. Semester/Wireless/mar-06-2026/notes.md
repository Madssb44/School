# NAT
Network Address Translation 
2 main types

## DNAT
Destination Network Address Translation is the translation protocol used when its the destination that needs to be translated  


## SNAT 
Source Network Address Translation is the translation protocol used when its the source needs to be translated

3 types
### Static NAT 
you have 1 public and 1 private address 

### Dynamic NAT 
when a network has more than 1 avaliable public ip address which can be use by the hostes 

### PAT NAT 
Port address transtation is when its port registred table
