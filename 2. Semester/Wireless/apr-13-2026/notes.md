## Communication modes
Theres a few communication modes we can use in a wireless network heres a few

### Infrastructure mode (BSS)
Infrastructure mode is the most commonly used in practice

it works by having a point everyone is communicating through such as a router or access point
so by having a access point control whos allowed to "talk"  you can avoid conflict while sending wireless communication

when you access a access point offering this you are then connected via a Basic Service Set or BSS and will communicate through it
you will also be talking via the MC address from the access point your are accessing to communicate via 

The area covered by the access point is often refered to as a Basic Service Area or BSA
All devices must be within the BSA to communicate wirelessly

Most access points can advertice up to 4 networks


Its also possible to extend a BSA this is called Extended Service Set or ESS

This allows roming between the different access points without disturbing your communication wirelessly

your BSSID will then be set depending on the access point you are on but the ESSID will be the same


### Ad-Hoc mode 
Independent Basic Service Set

This method requires you to manually configure the settings yourself but will allow for communication, but quictly gets degrades
the more users that are using the same configurations

so the effective bit rates are gonna fall drasticly the more users there are on it 

### Bridge mode 
Bridge mode is when 2 wiresess access points are connected and only send and accept communication from eachother
They commonly use a antenna and a parabole so direct the communication to avoid interference
