# DNS
when you make a dns query it goes through some steps before you get your destivation

first is the root server this is where the . is handled 

then the top level domain which is where the com, org, net etc is handled.

then you go out into the domain level where you will handle google, dr, youtube etc.

And lastly the subdomain level if needed where you will handle things like docs.google, mail.google etc.
