The programming exam will consist of different tasks that we are gonna write some code for in python, theres different types of tasks that change the starting point of how we are gonna write the code.

In this type we will be given a flowchart with some text and form that we are gonna have to write a program that follows the flowchart while completing the different tasks specified 

### **Text problem**
In this type we are given some tasks to compleat written in text format with no other direction on how to solve the problems and then we have to write a program to execute the tasks specified
